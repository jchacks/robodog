# Robodog

## Installation

- `apt update`
- `apt upgrade`
- Optional: `sudo apt install -y python3-pyqt5 python3-opengl`
- `python3.9 -m venv .venv`
- `source .venv/bin/activate`
- `pip install --upgrade pip`
- `pip install ".[dog]"`
